# HCIA-Datacom

### Course Contents
1) Huawei VRP Basics
      - Basic Device (Router, Switch) Configuration;
      - Remote Access (Telnet, SSH).
2) IP Routing
      - Static Routing;
      - Dynamic Routing. OSPFv2.
3) Virtual LAN (VLAN)
4) Spanning Tree Protocol (STP)
5) Link Aggregation. Eth-Trunk
6) Access Control List (ACL)
7) AAA (Authentication, Authorization and Accounting)
      - Local AAA;
      - Remote AAA (HWTACACS, RADIUS).
8) Network Address Translation (NAT)
9) Network Services (DHCP, HTTP, DNS, NTP, FTP, TFTP)
10) Wireless LAN (WLAN)
11) WAN Technologies. HDLC and PPP
12) IPv6 Addressing and OSPFv3
13) Network Automation. Python
14) Network Automation. Ansible
15) ReView
